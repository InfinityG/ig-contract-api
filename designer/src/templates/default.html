Contract Designer


Features

The Smart Contract Designer is used to create smart contracts in JSON format, for use by our Smart Contract API. Features include:

  • Template creation
  • Contract creation based on saved templates
  • Contacts required for your contracts are automatically retrieved from your ID-IO profile

Start here

  1. Create a template in the Template Designer and save it
  2. Choose a saved template from My Templates and then 'create a contract'
  3. Complete your contract by completing the empty fields and dragging in your contacts
  4. View the resulting JSON output at any time by clicking ''Show JSON' at any point
